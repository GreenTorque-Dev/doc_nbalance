# Health Check Logs

Health check logs in nBalance are records or entries that provide information about the status 
and results of health checks performed by the load balancer. These logs help monitor the 
health and availability of backend servers or services. 
##  User Activity logs 
User activity logs are records that track and store information about actions performed 
by users within the nBalance.
