### Management IP 

1. **IP Address:** IP addresses serve the purpose of identifying and locating devices on a 
network. 
2. **Gateway:** It will help to connect different networks 
3. **DNS Server:** It will help to convert the domain name into IP address. 
Click on> **Update** button will update the specific field. 
Click on> **Add** button will add the DNS server. 
Click on> **Delete** icon Button will delete the DNS server.
