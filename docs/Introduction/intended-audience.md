# Intended Audience 

This document is designed to assist individuals seeking guidance on configuring the nBalance 
system through the Web User Interface (WUI). Whether you are a novice or an experienced user, 
the information provided here serves as a helpful resource for anyone looking to customize and 
fine-tune the settings of the nBalance platform using the intuitive WUI.
