# Web Application Firewall (WAF) ​with nBalance​

​
A Web Application Firewall protects web applications by filtering, monitoring, and blocking malicious HTTP traffic.​
​
## nBalance includes WAF features to help protect against: 

- SQL Injection
- Cross-site scripting (XSS)
- Path traversal Bot attacks & automated threats
- Application-layer DDoS

## How WAF works:​

nBalance inspects incoming traffic before it reaches your
application and blocks anything suspicious, based on security rules
and patterns

### How it Works

1. Client sends a request to your website/app
2. nBalance receives the request
3. It checks which server is healthy and least loaded
4. Forward the request to the optimal server
5. Response is returned seamlessly to the client
