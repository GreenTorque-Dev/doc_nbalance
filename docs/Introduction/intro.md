# nBalance  
**Green Torque Technologies LLC**

Welcome to **nBalance**, a powerful Load Balancer and Web Application Firewall (WAF) designed to optimize the distribution of application traffic across multiple servers while ensuring robust security measures.

The primary goal of nBalance is to eliminate performance bottlenecks by preventing any single server from shouldering an excessive load. This not only enhances system reliability but also significantly improves overall availability.

---

# 🚀 Features

1. Bot Protection  
2. IP Persistence  
3. SSL Offloading  
4. Session Control  
5. Content Caching  
6. Web Acceleration  
7. Cookie Persistence  
8. Connection Control  
9. Session Persistence  
10. DDoS Attack Protection  
11. Layer 7 & Layer 4 Load Balancing  
12. Open Relay Protection (SMTP)  
13. GEO Restriction (Geo-based traffic control)  
14. Block admin URLs exposed to the internet  
15. Block PowerShell scripts (remote execution)  
16. Server Down Notifications  
17. Traffic and Session Graphs  
18. Multiple Backend Support  
19. Unit Test Coverage
