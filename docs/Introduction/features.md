### ​Layer 4 & Layer 7 Load Balancing​

Whether your client needs to split traffic evenly or prioritize certain
users (like premium ones), nBalance makes sure the right traffic goes
to the right server—faster performance and better customer
experience.​

![Layer 4 & Layer 7 Load Balancing](/img/layer.png)

### Application Health Checks

Constantly checks if servers are working properly and only sends
traffic to healthy ones.​
The client gets notified if any server goes down via email.​
Your client never has to worry about downtime. If one server goes
down, nBalance automatically shifts traffic to the working ones. Their
customers stay online, always.​

### IPv6/v4 Compatibility​​

Supports both modern and older internet protocols.​
No matter what devices or networks their customers are on, ​
nBalance ensures compatibility and accessibility—future-proofing their
app.​

## SSL Offloading

nBalance handles the website encryption, so your client’s servers
don’t have to.​

![SSL Offloading](/img/ssl.png)

### SSL Offloading

**Speed** – Sites load faster because servers don’t consume
CPU/memory for encryption. 
**Security** – Keeps customer data encrypted and safe. 
**Efficiency** – Frees up server power for actual business tasks

​### Reverse Proxy​

A reverse proxy is a server that sits in front of your web servers,
handling all client requests and forwarding them to the right
backend. 

- Every incoming connection first hits nBalance’s reverse proxy engine.
- Based on rules and real-time performance, it forwards requests to the best-fit backend server
